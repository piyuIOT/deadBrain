# Motivation
## What is the session about?
- The session is to learn to use Claude Code primarily for programming related tasks, which includes but is not restricted to:
	- Writing new code
	- Restructuring old code
	- Understanding code
	- Making documentation
	- Refactoring
- Connecting Claude Code with external tools (MCP) like your Browser, Github, Postman, etc.
- Agentic orchestration
- Parallel programming with Claude code
- Understanding: Tools, Function calls, Agents, Skills, MCP, Agentic orchastration
- Idea about different mode of working (thinking, planning, execution etc)

## What the session is definitely not about?
1. Philosophical discussion onto whether or not we should use AI in proprietary codebase?
	- Better first we develop "Proprietary codebase" which is worth enough to be stolen
2. Whether or not our code is used for training of AI or not?
	- There is a setting to prevent AI training on your data. I have no information on its effectiveness.
3. Is AI coding good for you as a developer or not?
	- Without it you are going out of business
	- Rely only on AI and you are again going out of business
	- Always, always understand what the AI is doing, don't leave it on steroids to get your job done, otherwise, you are again out of business, we would rather hire the AI itself.
	- Better learn the symbiosis and you might survive
