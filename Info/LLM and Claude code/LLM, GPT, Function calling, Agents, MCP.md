# Motivation
## What this session is about?
- Understanding how and Language model works
- Understanding the GPT architecture
- Understanding Function Calling
- Understanding Agents
- Understanding Skills
- Understanding Model context protocol (MCP)
- Understanding how using Agents in your development pipeline can help you
## What this session is not about?
- Mathematics behind neural nets, LLMs, GPTs
- Philosophical discussions on LLMs, their implications, their side effects, safety concerns and related fields
- Live coding LLM or related systems
